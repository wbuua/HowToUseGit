1.配置git
#查看信息
$ git config --global --list  
$ git log --oneline
$ git status
#设置git用户名、密码、邮箱的配置（全局配置）
$ git config --global user.name 用户命
$ git config --global user.password 密码
$ git config --global user.email 邮箱

2..初始化文件夹git仓库
$ git init

3.添加文件到暂存区，提交文件到本地仓库
$ git add .   将所有更改添加到暂存区
$ git commit -m “备注信息”  将暂存区的文件提交到本地仓库

5.pull和push
